Примечание

Ниже указаны CSS свойства align-self, align-content, flex-shrink, flex-basis.
Для каждого свойства указано 2 примера: в первом основной осью является горизонталь, во втором основной осью является вертикаль.
Посмотреть, как они влияют с теми или иными параметрами можно в файле assets/css/styles.sass, раскомментив нужный параметр.

Align-self

Свойство align-self - управляет выравниванием конкретного flex элемента по поперечной (второстепенной) оси в пределах области контерйнера, в котором он задан.
Значение по умолчанию: auto / указывать самому не нужно.
Auto - берёт родительское значение align-items или если нет родителя принимает значение stretch (занимает все свободное пространство контейнера по поперечной оси
align-self: flex-start // выбранный элемент выравнивается к началу поперечной оси
align-self: center // выбранный элемент выравнивается к центру (середине) поперечной оси
align-self: flex-end // выбранный элемент выравнивается к концу поперечной оси
align-self: baseline // выбранный элемент выравнивается по базовой линии текста (то есть располагается на уровне первой строки в контейнере).
align-self: stretch // выбранный элемент растягивается таким образом, чтобы занять всё свободное пространство контейнера по поперечной оси.

Align-content

Свойство align-content - задаёт тип выравнивания строк внутри flex контейнера по поперечной (второстепенной) оси при наличии свободного пространства.
Значение по умолчанию: stretch
Данное свойство отличается от align-items тем, что align-content определяет расстояние между строками, а align-items определяет как элементы в целом выровнены внутри контейнера. Когда есть только одна строка, align-content не работает
align-content: flex-start // Строки располагаются в начале поперечной оси. Каждая следующая строка идёт вровень с предыдущей.
align-content: center // Строки располагаются по центру контейнера.
align-content: flex-end // Строки располагаются начиная с конца поперечной оси. Каждая предыдущая строка идёт вровень со следующей.
align-content: space-between // Строки равномерно распределяются в контейнере и расстояние между ними одинаково.
align-content: space-around // Строки равномерно распределяются таким образом, чтобы пространство между двумя соседними строками было одинаковым. Пустое пространство перед первой строкой и после последней строки равно половине пространства между двумя соседними строками.
align-content: space-evenly // Строки распределяются равномерно. Пустое пространство перед первой строкой и после последней строки имеет ту же ширину, что и у других строк.
align-content: stretch // Строки равномерно растягиваются, заполняя свободное пространство.

Flex-shrink

Свойство применяется только в том случае, если размер элементов в флекс-контейнере вызывает переполнение флекс-контейнера
Свойство flex-shrink задает коэффициент сжатия flex с заданным числом. Отрицательное значение не валидно.
Значение по умолчанию: 1.

Flex-basis

Свойство flex-basis определяет основу флекса, которая является начальным размером элемента. Похоже на свойства width и height, к которым добавляется содержимое элемента.
Значение по умолчанию: auto (берет размеры из width и height)
